What is the Java Samples Kit?

Use this client side SDK to build security solutions for any Java Based environment.

How does it work?

The library supports all the standard Secplugs functionality allowing access to file web and email functionality and makes it easier than integrating direclty with the REST APIs.

How do I get started?

To get started download the library, follow the instructions to get it installed and then take a look at the examples.

To use additional features and the privacy of your own account, after registering in Secplugs.com, login with your username and create an API key to use with the scripts. Replace the key in the samples or create new scripts using these as an example. Use can then use the Secplugs console to view activity, run reports and do deeper retrospective threat analysis.
